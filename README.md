# coding-agent-preferences

This repository contains my personal preferences for coding, deployment, and development environments. These documents are specifically designed for AI coding agents to follow when assisting in a project.

The contents of these files can be used to kickstart new projects by providing clear, concise, and opinionated instructions on how I like to work.

## Index of Preferences

- [AGENTS.md](AGENTS.md) - Coding philosophy, conventions, and minimalism principles for Python, Bash, and general software engineering.
- [DEPLOYMENT.md](DEPLOYMENT.md) - Preferred patterns for web app deployments, including Linux service isolation, directory layouts, and configuration templates for nginx and systemd.
